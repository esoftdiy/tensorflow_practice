预测时的数据准备：
You have to create a PNG file that contains a handwritten number.
The background has to be white and the number has to be black.
Any paint program should be able to do this. Also the image has to be auto cropped so that there is no border around the number.

28*28像素，白底黑子图片（jpg，png图像都可以），注意，白底黑子
